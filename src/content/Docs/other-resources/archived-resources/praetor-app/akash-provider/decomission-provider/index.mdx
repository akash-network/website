##  Single Server Provider


To execute the following commands, first SSH into your machine using the credentials you used for Praetor.


`/usr/local/bin/k3s-uninstall.sh`

and:

`rm -rf ~/.akash/ ~/bin/ ~/.kube/ ~/.praetor/ ~/ingress-nginx-custom.yaml`
