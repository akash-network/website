**Deploy and manage applications on Akash Network using the web-based Akash Console.**

Akash Console ([console.akash.network](https://console.akash.network)) is a visual web interface for deploying on Akash without CLI tools. Use it with a free trial (credit card) or your own crypto wallet.

---

## Getting Started

### [Getting Started with Console](/docs/developers/deployment/akash-console/getting-started)

Learn the basics of Akash Console, including:
- Console interface and features
- Visual SDL builder
- Deployment dashboard
- Common tasks and workflows
- Console vs CLI comparison

**Start here if you're new to Akash Console.**

## Deployment Options

### [Using Console with Your Wallet](/docs/developers/deployment/akash-console/with-wallet)

Deploy using your own Keplr or Leap wallet for full control:
- Connect your crypto wallet
- Create and manage deployments
- No time limits on deployments
- Direct blockchain access

**Use this if you have AKT and want production deployments.**

---

### [Managed Wallet API](/docs/developers/deployment/akash-console/api)

Programmatic deployments using the Console API:
- REST API for automated deployments
- JWT authentication
- Pay with credit card
- No wallet management required

**Use this for automated deployments via API.**
