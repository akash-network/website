**Deploy and manage Akash applications using our official SDKs in Go or JavaScript/TypeScript.**

Both SDKs are generated from the chain-sdk and provide the same core functionality.

---

## Quick Start

### 1. Import and Initialize

<CodeTabs
  client:load
  examples={[
    {
      language: "go",
      code: `import (
    "github.com/akash-network/akash-go/client"
    "github.com/akash-network/akash-go/wallet"
)

// Initialize client
akashClient := client.NewClient(client.Config{
    Network:  "mainnet", // or "testnet"
    NodeURL:  "https://rpc.akash.forbole.com:443",
})

// Load wallet from mnemonic
wallet, err := wallet.FromMnemonic("your mnemonic phrase here")
if err != nil {
    log.Fatal(err)
}
// Or from key file
// wallet, err := wallet.FromKeyFile("path/to/key.json")`,
    },
    {
      language: "javascript",
      label: "JavaScript/TypeScript",
      code: `import { AkashClient } from '@akashnetwork/chain-sdk';
import { Wallet } from '@akashnetwork/chain-sdk/wallet';

// Initialize client
const client = new AkashClient({
  network: 'mainnet', // or 'testnet'
  nodeUrl: 'https://rpc.akash.forbole.com:443'
});

// Load wallet from mnemonic
const wallet = Wallet.fromMnemonic('your mnemonic phrase here');
// Or from key file
// const wallet = Wallet.fromKeyFile('path/to/key.json');`,
    }]}
/>

### 3. Accept a Bid and Create Lease

<CodeTabs
  client:load
  examples={[
    {
      language: "go",
      code: `// Wait for bids (or check existing bids)
bids, err := client.Market.GetBids(ctx, wallet.Address(), deployment.DSEQ)
if err != nil {
    log.Fatal(err)
}

// Accept the first bid
if len(bids) > 0 {
    selectedBid := bids[0]
    lease, err := client.Market.CreateLease(ctx, wallet, deployment.DSEQ, selectedBid.Provider)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Lease created with provider: %s\\n", lease.Provider)
}`,
    },
    {
      language: "javascript",
      label: "JavaScript/TypeScript",
      code: `// Wait for bids (or check existing bids)
const bids = await client.market.getBids({
  owner: wallet.address,
  dseq: deployment.dseq,
});

// Accept the first bid
if (bids.length > 0) {
  const selectedBid = bids[0];
  const lease = await client.market.createLease({
    wallet,
    dseq: deployment.dseq,
    provider: selectedBid.provider,
  });
  console.log(\`Lease created with provider: \${lease.provider}\`);
}`,
    }]}
/>

## Core Features

All SDKs support:

- ✅ **Deployment Management** - Create, update, close deployments
- ✅ **Market Operations** - View bids, create leases
- ✅ **Manifest Management** - Send and update manifests
- ✅ **Query Operations** - Query deployments, leases, providers
- ✅ **Wallet Management** - Load wallets, sign transactions
- ✅ **SDL Parsing** - Parse and validate SDL files
