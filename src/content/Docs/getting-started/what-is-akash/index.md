**Akash Network is the world's first decentralized cloud computing marketplace.** Think of it as Airbnb for cloud computing‚Äîpeople with spare computing power can rent it out to those who need it.

## In 60 Seconds

Instead of paying AWS, Google Cloud, or Azure for servers, you can:
- Deploy your applications on a global network of independent providers
- Pay **up to 85% less** than traditional cloud providers
- Get **censorship-resistant** infrastructure
- Support a **decentralized** internet

**It's real cloud computing, but open, permissionless, and affordable.**

---

## How It Works

### 1. **You Deploy**
Create a deployment using our CLI, Console, or SDK. Define what you need (CPU, RAM, storage) in a simple YAML file.

### 2. **Providers Bid**
Independent cloud providers around the world compete to host your application by submitting bids.

### 3. **You Choose**
Pick the best offer based on price, location, and provider reputation.

### 4. **Your App Runs**
Your application runs on the provider's infrastructure. You pay per block, and can close anytime.

## Key Concepts

### Deployment
A **deployment** is your application definition‚Äîwhat resources you need and how your application should run.

### Provider
A **provider** is an independent cloud operator offering computing resources on Akash.

### Bid
A **bid** is an offer from a provider to host your deployment at a specific price.

### Lease
A **lease** is the active agreement between you and a provider. Once accepted, your deployment runs on their infrastructure.

### SDL
The **Stack Definition Language (SDL)** is a YAML format that describes your deployment‚Äîsimilar to Docker Compose.

### AKT
**AKT** is Akash's native cryptocurrency used for payments, staking, and governance.
